HTML Sigla para HyperText Markup Language — Linguagem de Marcação de Hipertexto —, o HTML é o componente base da web. Isso quer dizer que ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos.

O que são TAGS?

Que são tags HTML? Essas tags definem o conteúdo e a formatação de um documento HTML. Elas são usadas para estruturar o conteúdo de um documento, para formatar o texto, para adicionar imagens e outros elementos, etc.

As tags HTML básicas são essenciais para criar qualquer documento HTML.

H1 até H5 - Cabeçalho

As tags de cabeçalho são responsáveis por definir o cabeçalho principal de uma página (H1), assim como os subcabeçalhos, ou subtítulos, (H2 – H6) de outras seções de conteúdo. A importância das headings tags, ou seja, os títulos e subtítulos presentes num conteúdo, está relacionada à experiência do usuário.
